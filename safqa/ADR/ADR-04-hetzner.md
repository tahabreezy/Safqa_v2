# ADR-04: Hetzner VPS Hosting

## Context
The MVP requires self-hosted infrastructure under strict monthly cost constraints.

## Decision
Deploy on a single Hetzner CX32 VPS.

## Rationale
Hetzner provides low-cost resources, root control, and predictable operating model.

## Rejected Alternatives
- Railway
- Render

## Consequences
- Team owns server hardening, monitoring, and operations.
- Cost target is achievable for launch.
